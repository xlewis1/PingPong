Ping Pong (PONG) Web App

A simple, browser-based Ping Pong (PONG) game that works on desktop, tablet, and mobile devices. Play against yourself or with a friend using keyboard or touch controls. Built with HTML, CSS, and JavaScript.

Features
	•	Classic Pong gameplay
	•	Works on desktop, tablet, and mobile
	•	Keyboard controls for desktop:
	•	Player 1: W / S
	•	Player 2: Up / Down arrow keys
	•	Touch controls for mobile devices
	•	Score tracking
	•	Responsive canvas that adapts to different screen sizes
	•	Lightweight and fast

How to Play
	1.	Open the site in a browser.
	2.	Use the controls to move the paddles and hit the ball.
	3.	First player to reach the set score wins.
	4.	On mobile, use the touch controls displayed under the canvas.
